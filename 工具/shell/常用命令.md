# 常用命令

查看进程由那个命令启动

```
systemctl status pid  
```

#### 查看打开句柄总数

```
lsof|awk '{print $2}'|wc -l
```

#### 根据打开文件句柄的数量降序排列，其中第二列为进程ID

```
lsof|awk '{print $2}'|sort|uniq -c|sort -nr|more
```

#### 查看某个进程已经打开的文件数

```
cat /proc/$pid/limits
```
